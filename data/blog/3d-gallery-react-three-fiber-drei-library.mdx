## Hello! Welcome to my first blog post.

Discovering the World of 3D Web Graphics
A couple of months ago, I worked on a small project and I'm excited to tell you about it. Let's start from the beginning.

I've been a web developer for over 10 years. I've done a lot with websites and animations, but I wanted to try something new. My goal is always to make websites with appealing animations that make them more fun and lively.

## Inspiration from Robin Noguier's Innovative Design

While looking for ideas, I came across this awesome website by Robin Noguier ([check it out here](https://robin-noguier.com/)). Robin Noguier is a highly skilled web designer known for his unique and captivating website designs. His work is characterized by smooth animations and innovative use of modern web technologies, combining artistic flair with technical prowess.

The website I found had animations that were really different from what I usually see, showcasing a blend of creativity and technical skill. It was developed using Vercel's Next.js and WebGL, both cutting-edge technologies in the web development world.

## My Dive into WebGL and React Three Fiber

I got really interested in WebGL, which is all about making special 2D and 3D graphics that work in web browsers. It's a bit tricky, but really powerful for making graphics come alive on websites.

Then I found this awesome tool called [React Three Fiber](https://github.com/pmndrs/react-three-fiber). It's a special way to use Three.js (a tool for 3D graphics) with React, which is something I already know well. Basically, it lets me create 3D stuff on websites using React, which is super handy.

I also stumbled upon [Drei](https://github.com/pmndrs/drei) from the pmndrs community. It's like a helper for React Three Fiber, making it easier to add things like lights, shapes, and better-looking materials in 3D scenes. It's great because it saves time and makes working with 3D graphics a lot simpler.

With these tools, I could mix my React skills with awesome 3D graphics, making web development even more fun and creative.

## Crafting a 3D Gallery: A Creative Exploration

<div className="w-full">![3d-gallery](/static/images/blog/3d-gallery.jpeg)</div>

After playing around with these tools, I found a cool [example of a 3D Gallery](https://codesandbox.io/p/sandbox/image-gallery-lx2h8?file=%2Fsrc%2Fstyles.css). It showed me how to use something called "MeshReflectorMaterial" from the pmndrs/drei library. It's just an example of how to use reflecting materials on shapes with different options.

I liked the clean style of the project and decided to add some additional features, such as:

- Navigating through pictures with arrows.
- Adding custom routing with "wouter" to share art items.
- Providing descriptions for each piece of art.
- Offering the option to post a video instead of an image.

You can find a Demo [here](https://3d-gallery-eta.vercel.app/) <br />
And the Repo [here](https://github.com/eugenregehr/r3f-image-gallery)

I'm still learning WebGL, especially the topic of "Shaders," which is quite challenging. But I'm determined to keep trying. Following this project, I've developed several other projects using React Three Fiber (R3F) that I'm excited to share with you.
